# NROPT

```
NROPT, Option1, Option2, Optval
Specifies the Newton-Raphson options in a static or full transient analysis.
```

- Option1

  + AUTO
  
        由程序自主选择(默认)

  + FULL
  
        完全Newton-Raphson

  + MODI
  
        修正Newton-Raphson
        
  + INIT
  
        使用以前的计算矩阵(初始刚度)
