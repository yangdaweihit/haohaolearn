# Visual Studio Code

## 常见快捷键

| 快捷键       | 功能         |
|--------------|--------------|
| Ctrl+,       | 设置面板     |
| Ctrl+Shift+p | 命令面板     |
| Ctrl+Shift+x | 插件面板     |
| Ctrl+b       | 关闭左侧面板 |


## python快捷键


| 快捷键       | 功能         |
|--------------|--------------|
| Ctrl+Shift+i | 格式化代码   |
