INKSCAPE

2020-3-18

简介

  • Inkscape是一款开源矢量绘图软件，适合在科研中实现示意图的绘制
  • 一款全平台软件，适用于 GNU Linux/Windows/macOS
  • 当前稳定版：0.92.4
  • 最新开发版：1.0
  • 官网下载地址：https://inkscape.org/release/inkscape-0.92.4
